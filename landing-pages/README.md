# PersonaForge Landing Pages

This directory contains generated landing pages for each creator/model.

## Structure

```
landing-pages/
├── ellie-mayees/           # Creator folder (username slug)
│   ├── coaching-1.html     # Coaching page variant 1
│   ├── coaching-2.html     # Coaching page variant 2
│   ├── quiz-1.html         # Quiz page variant 1
│   └── quiz-2.html         # Quiz page variant 2
├── another-creator/
│   └── ...
```

## Page Types

- **Coaching**: Free session/clarity call landing pages
- **Quiz**: Interactive quiz landing pages with lead capture

## Deployment

Each creator's pages can be deployed to their own domain:
- Primary: `https://creatorname.com/coaching-1`
- Edge: `https://creatorname.pages.dev/quiz-1`
